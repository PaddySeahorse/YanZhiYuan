# Complete API

This section provides the complete API reference for Norse.

## norse

The top-level norse package containing all submodules:

- `norse.torch` - Main module with PyTorch-compatible building blocks
- `norse.torch.functional` - Functional implementations
- `norse.torch.module` - Neural network modules
- `norse.torch.utils` - Utility functions
- `norse.dataset` - Dataset loaders
- `norse.task` - Task examples
- `norse.benchmark` - Benchmarking tools
